OpenAI 앱 제출 기능 및 AI 개발 가속

4 min read · Dec 28, 2025

1. AI/기술 트렌드

a. AI 모델 & 서비스

  • OpenAI, ChatGPT 내 개발자 앱 제출 허용: 개발자가 직접 만든 앱을 ChatGPT에 제출하고 게시할 수 있는 디렉토리가 출시되어, 사용자가 승인된 앱을 쉽게 발견할 수 있게 되었습니다.
  • 새로운 ChatGPT Images 출시: 플래그십 이미지 생성 모델을 통해 더욱 정교한 편집, 일관된 디테일 유지 및 최대 2배 빠른 생성 속도를 제공합니다.
  • Apple, 2D 사진의 3D 변환 모델 공개: 2D 사진을 즉시 3D 뷰로 변환하는 오픈소스 모델 ml-sharp를 GitHub에 공개했습니다.

b. AI 기업 동향

  • OpenAI Academy 출범: 뉴스 조직을 위한 학습 허브를 런칭하여 저널리즘 분야의 AI 활용을 지원합니다.
  • BNY의 전사적 AI 도입: BNY(Bank of New York Mellon)는 OpenAI 기술을 활용한 'Eliza' 플랫폼을 구축하여 2만 명 이상의 직원이 AI 에이전트를 활용하도록 확장했습니다.
  • Anthropic 연구 업데이트: 'Project Vend'의 두 번째 단계를 공개하며 AI 연구 및 개발의 새로운 진척 상황을 공유했습니다.

c. AI 연구 & 기술

  • FrontierScience 벤치마크 도입: 물리학, 화학, 생물학 분야에서 AI의 추론 능력을 측정하여 실제 과학 연구 수행 가능성을 평가하는 벤치마크를 출시했습니다.
  • AI를 활용한 생물학 연구 가속: GPT-5를 활용해 습식 실험실(Wet Lab)에서의 분자 최적화 등 생물학 연구 속도를 높이는 평가 프레임워크를 발표했습니다.

2. 개발 업데이트

a. Vibe Coding

  • Vibe-Kanban 등장: AI 코딩 에이전트를 효율적으로 관리하기 위한 Rust 기반의 칸반 보드 프로젝트가 GitHub에서 주목받고 있습니다.
  • Claude Code 오케스트레이션: wshobson/agents 프로젝트를 통해 Claude Code를 기반으로 한 멀티 에이전트 오케스트레이션과 지능형 자동화 구현 사례가 확산 중입니다.

b. AI 모델 & API

  • GPT-5.2-Codex 안전 사양 공개: 모델 수준의 완화 조치와 특화된 안전 시스템을 포함한 GPT-5.2-Codex의 시스템 카드가 상세히 공개되었습니다.
  • Gemini CLI 활용: Google AI는 Gemini CLI를 활용한 에이전트 팩토리 구축 및 딥다이브 내용을 공유하며 개발자의 CLI 접근성을 강화했습니다.

c. 개발 트렌드

  • Python 패키지 매니저 'uv'의 부상: Rust로 작성된 uv가 압도적인 속도를 낼 수 있었던 아키텍처적 비결에 대한 분석이 개발자 사이에서 큰 인기를 끌고 있습니다.
  • Rust 기반 커널 확장 'Rex': eBPF를 대체하여 Rust로 안전한 커널 확장을 작성할 수 있게 해주는 프레임워크 Rex가 공개되었습니다.
  • Web 표준의 회귀: JavaScript를 최대한 HTML로 대체하려는 시도(HTMHELL)와 Sass를 떠나 표준 CSS로 전환하는 흐름이 감지되고 있습니다.

3. AI Coding Assistant

  • Codex를 통한 초고속 개발 사례: OpenAI 팀은 Codex와 AI 지원 병렬 코딩 워크플로우를 활용하여 단 28일 만에 'Sora for Android' 앱을 개발하고 출시했습니다.
  • Cloudflare Agents: Cloudflare 환경에서 AI 에이전트를 직접 구축하고 배포할 수 있는 도구가 출시되어 에이전트 개발 장벽이 낮아졌습니다.
  • Easel IDE: Clojure 언어로 구축된 자체 IDE인 Easel이 1주년을 맞이하며 에디터 시장의 지속적인 실험적 시도를 보여주었습니다.
  • Type-Safe API 설계: itty-spec과 같은 도구를 사용하여 AI 시대에 더욱 중요해진 Contract-first 방식의 타입 안전 API 구축 방법이 공유되고 있습니다.

  • 링크: https://yossarian.net/til/post/serde-s-borrowing-can-be-treacherous/

5. 인사이트

최근 트렌드는 AI 에이전트가 단순한 '코드 보조'를 넘어 '실제 제품 출시(Sora Android 28일 완성)'와 '에코시스템 구축(ChatGPT 앱 스토어)' 단계로 급격히 진화하고 있음을 보여줍니다. 동시에 개발 생태계에서는 uv나 Rex와 같이 성능과 안전성을 극대화하기 위해 Rust를 적극 도입하는 경향과, 복잡한 프레임워크 대신 HTML/CSS 표준으로 회귀하여 유지보수성을 높이려는 실용주의적 접근이 공존하고 있습니다. 특히 AI 에이전트 간의 협업(Multi-agent) 관리 도구들이 등장하기 시작한 점은 향후 개발 워크플로우의 중심이 에이전트 오케스트레이션으로 이동할 것임을 시사합니다.---
