AI 에이전트 확산과 바이브 코딩

4 min read · Dec 30, 2025

1. AI/기술 트렌드

a. AI 모델 & 서비스

  • OpenAI FrontierScience 발표: 물리, 화학, 생물학 분야에서 AI의 과학적 추론 능력을 측정하는 새로운 벤치마크를 도입했습니다.
  • ChatGPT 이미지 기능 업데이트: 자사 플래그십 모델을 통해 더욱 정밀한 편집과 일관된 디테일을 제공하는 이미지 생성 기능을 출시했습니다.
  • GPT-5.2-Codex 시스템 카드 공개: 모델 수준의 완화 조치와 특수 안전 프로토콜을 포함한 상세한 안전 가이드라인을 발표했습니다.
  • ChatGPT 앱 디렉토리 런칭: 개발자들이 직접 만든 앱을 ChatGPT 내부에 제출하고 사용자가 탐색할 수 있는 공식 디렉토리가 열렸습니다.

b. AI 기업 동향

  • Meta, AI 에이전트 스타트업 'Manus' 인수: 최근 업계에서 화제가 된 AI 에이전트 기업 Manus를 인수하여 Facebook, WhatsApp 등에 통합할 계획입니다.
  • OpenAI Academy 설립: 언론사를 대상으로 AI 활용 교육을 제공하는 허브를 구축하여 미디어 산업과의 파트너십을 강화합니다.
  • AI 칩 가격 상승 경고: AI 열풍으로 인한 메모리(RAM) 수요 급증이 전자기기 전반의 가격 인상을 초래할 수 있다는 분석이 제기되었습니다.

c. AI 연구 & 기술

  • Z80-μLM 경량화 AI: 단 40KB 메모리 환경에서 구동되는 대화형 AI 프로젝트가 공개되어 임베디드 AI의 가능성을 보여주었습니다.
  • 생물학 연구 가속 프레임워크: GPT-5를 활용해 습식 실험실(Wet Lab) 내 분자 최적화 등 연구 속도를 높이는 평가 프레임워크가 제시되었습니다.

d. AI 규제 & 정책

  • AI 생성 가짜 뉴스 주의: 영국의 키어 스타머 총리를 타겟으로 한 AI 합성 가짜 영상이 틱톡에서 확산되는 등 규제와 검증의 중요성이 커지고 있습니다.

2. 개발 업데이트

a. Vibe Coding

  • 바이브 코딩(Vibe Coding)의 확산: Claude Code 등을 활용해 직관적으로 코딩하는 트렌드가 주목받는 동시에, "LLM 코딩은 재미없다"는 비판적 시각도 공존하고 있습니다.
  • vibe-kanban 프로젝트 인기: Claude Code나 Codex 같은 코딩 에이전트의 효율을 10배 높여주는 칸반 스타일의 관리 도구가 GitHub에서 급부상했습니다.

b. AI 모델 & API

  • Postgres 전용 MCP 서버: PostgreSQL 숙련도와 문서를 학습시켜 AI 코딩 도구가 더 나은 SQL 코드를 생성하도록 돕는 pg-aiguide가 공개되었습니다.
  • 실시간 대화 에이전트 구축: LiveKit과 Gemini, Express를 조합해 실시간 음성 대화가 가능한 AI 에이전트를 구현하는 가이드가 공유되었습니다.

c. 개발 트렌드

  • Curl 보안 강화: 유명 네트워크 도구인 Curl이 보안을 위해 코드 내 모든 strcpy 함수 호출을 제거하고 안전한 대안으로 교체했습니다.
  • Rust 가비지 컬렉션 개선: Rust의 가비지 컬렉션을 보완하는 라이브러리 dumpster2가 업데이트되어 메모리 관리 효율이 높아졌습니다.
  • Bun의 성능 우위: Node.js보다 빠른 실행 속도를 보여주는 Bun 런타임에 대한 벤치마크와 관심이 지속되고 있습니다.

3. AI Coding Assistant

  • Claude Code 망각 방지 도구: Claude Code가 문맥을 잊어버리는 문제를 해결하기 위한 커뮤니티 도구인 ensue-skill이 GitHub에서 주목받고 있습니다.
  • Anthropic Agent Skills: Anthropic이 공개한 에이전트 전용 스킬 저장소를 통해 개발자들이 에이전트의 기능을 더 쉽게 확장할 수 있게 되었습니다.
  • AI 도구 시스템 프롬프트 모음: Cursor, Claude Code, Manus 등 주요 AI 도구들의 작동 원리를 파악할 수 있는 시스템 프롬프트 저장소가 인기를 끌고 있습니다.
  • Gitdocs AI: GitHub 저장소로부터 실제 배포 가능한 수준의 README 파일을 자동 생성해 주는 도구가 출시되었습니다.

5. 인사이트
